FFDecks

FFDecks is a Final Fantasy Trading Card Game deck builder, tournament results hub, meta analysis, playtesting platform, card database, collection manager, and community content hub.

This site was developed to fill the gap for the community's lack of full encompassing websites to foster community growth for the game.

  • This website was developed with angular and a backend written in python. This website is hosted on google app engine and uses google datastores and firestore.
  • Realized four different revenue streams.
  • Voted number one community website by the players of the game. Feature development was guided by community feedback.

Service Fabric Explorer

Service Fabric Explorer or SFX is an application for inspecting and managing cloud applications and nodes in a Microsoft Azure Service Fabric cluster.

SFX is used to monitor and diagnose service fabric clusters for high production work loads. Improvements made have generated significant time saving and cost for areas which previously made the product difficult to easily support.

  • This website was originally written in angularjs but has been migrated to angular. Testing is done using cypress to cover broad End to End scenarios.
  • This product is developed closely with feedback from support engineers, other software engineers and end users. The development cycle used is very reactive to new feature requests to always prioritize the highest impact items
  • SFX is developed to meet ay11 accessibility standards
